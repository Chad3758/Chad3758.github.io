##大富翁小遊戲

#規則:
-遊戲人數4人起始金額為10000
-每買一筆土地1000、房子加蓋500(最多5層)
-稅金
-買地/蓋房子
-回到/經過起始點+2000
-破產

#細部:
-動態骰子
-骰子不能連點
-確認視窗是否占地/蓋房
-破產(<0)
-遊戲結束(三人破產)

##程式碼內容:

#JS:
-開始點位
-骰點
-動態骰子
-連點
-把數字寫進流程控制台
-圖的狀態
-用空白圖片把前一個骰子蓋掉
-破產的判定
-玩家位置
-判斷回合
-結束條件
-宣布勝利
-沒有買土地沒有見房子也沒有踩到別人的地
-走的步數
-採到他人的地
-加蓋/買地/收稅
-棋子移動
-路過原點

#CSS:
-排列位置
-基礎框架外觀
-其他外觀設定


#html:
-圖片
-玩家欄位設定
-地圖框架
